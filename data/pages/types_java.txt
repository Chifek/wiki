====== Типы переменных в Java. ======

  * Целочисленные (к ним относятся byte, short, int, long)
  * С плавающей точкой (к ним относятся float, double)
  * Символы (char)
  * Логические (boolean)

----------------------------

  * boolean b = true;
  * byte by = 127;
  * char c = 'A'; (ASCII таблицa)
  * short s = 32767;
  * int i = 2_300_000;
  * long l = 20_000_000L; (L не обязательно)
  * float f = 8.2F;
  * double d = 8.5;
